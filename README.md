## ğŸš€ Getting Started (Local Setup)

1. Clone the repo:
    ```bash
    git clone https://github.com/ShubhamMarwade/DSA-PLAYGROUND.git
    cd DSA-PLAYGROUND
    ```

2. Install frontend dependencies:
    ```bash
    npm install
    ```

3. Setup backend:
    ```bash
    cd backend2
    npm install
    ```

4. Add your `.env` files:
    - `backend2/.env`
    - `src/.env`

5. Run the backend:
    ```bash
    node server.js
    ```

6. Run the frontend:
    ```bash
    npm start
    ```
